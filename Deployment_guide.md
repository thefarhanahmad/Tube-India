# Bideo Production Deployment Guide

## Server Information

### Domain

- Website: https://bideo.in
- API Base URL: https://bideo.in/api

### VPS Details

- OS: Ubuntu
- Process Manager: PM2
- Web Server: Nginx
- SSL: Let's Encrypt (Certbot)

### Important Paths

Backend Source:

```bash
~/Bideo/backend
```

Frontend Source:

```bash
~/Bideo/adminDashboard
```

Live Frontend:

```bash
/var/www/bideo
```

Nginx Config:

```bash
/etc/nginx/sites-available/bideo.in
```

PM2 Process:

```bash
bideo-server
```

---

# Initial Server Checks

Check PM2 Status:

```bash
pm2 list
```

Check Backend Logs:

```bash
pm2 logs bideo-server --lines 50
```

Check Nginx:

```bash
nginx -t
```

Reload Nginx:

```bash
systemctl reload nginx
```

# Backend Deployment

Whenever backend code changes:

### Step 1: Pull Latest Code

```bash
cd ~/Bideo

git pull origin main
```

### Step 2: Go To Backend

```bash
cd backend
```

### Step 3: Install Dependencies

```bash
npm install
```

### Step 4: Restart Backend

```bash
pm2 restart bideo-server
```

### Step 5: Verify

```bash
pm2 logs bideo-server --lines 50
```

Expected:

```text
Server running on port 5000
MongoDB Connected
```

---

# Frontend Deployment

Whenever admin dashboard changes:

### Step 1: Pull Latest Code

```bash
cd ~/Bideo

git pull origin main
```

### Step 2: Open Frontend Folder

```bash
cd adminDashboard
```

### Step 3: Install Dependencies

```bash
npm install
```

### Step 4: Build Production Version

```bash
npm run build
```

Expected:

```text
✓ built successfully
```

### Step 5: Replace Live Files

```bash
rm -rf /var/www/bideo/*

cp -r dist/* /var/www/bideo/
```

### Step 6: Reload Nginx

```bash
systemctl reload nginx
```

### Step 7: Verify

Open:

```text
https://bideo.in
```

# PM2 Commands

View Running Apps:

```bash
pm2 list
```

Restart Backend:

```bash
pm2 restart bideo-server
```

Restart All:

```bash
pm2 restart all
```

View Logs:

```bash
pm2 logs
```

Save Processes:

```bash
pm2 save
```

Enable Startup:

```bash
pm2 startup
```

Run the command generated by PM2.

Then:

```bash
pm2 save
```

Backend Only:

```bash

git pull origin main

cd backend

npm install

pm2 restart bideo-server
```

Frontend Only:

```bash
cd ~/Bideo

git pull origin main

cd adminDashboard

npm install

npm run build
```

```bash

npm install

cd ../adminDashboard
npm install
npm run build

rm -rf /var/www/bideo/*
```
